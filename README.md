# Solana Mini-dApp Scaffold

A high-quality, lightweight entry point for building Solana Web3 applications. This scaffold provides the essential logic to connect to the Solana blockchain, check balances, and sign transactions.

## Features
* **Wallet Connection**: Seamlessly integrates with Phantom, Solflare, and other SPL wallets.
* **Web3 Integration**: Uses `@solana/web3.js` for direct cluster interaction.
* **Modern Stack**: Built with React and Tailwind CSS for a professional look.

## Quick Start
1. Install dependencies: `npm install @solana/web3.js @solana/wallet-adapter-react @solana/wallet-adapter-react-ui`
2. Run the development server: `npm run dev`

## Deployment
Compatible with Vercel, Netlify, or GitHub Pages. Ensure your RPC URL is configured in your environment variables for production.
